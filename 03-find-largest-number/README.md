# Find largest number

Write a function that returns the largest number in an array.

**Example 1:**

```
Input: nums = [5, -5, 10, 13, 2, 19, 4];
Output: 19
```

**Example 2:**

```
Input: nums = [-2, 7, 4, -1, 3, 0, 4];
Output: 7
```

**Example 3:**

```
Input: nums = [-1, -3, -8, -1, -2, -4, -5];
Output: -1
```
